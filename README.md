# 🔐 Caesar Cipher (Python)

A simple implementation of the Caesar Cipher encryption and decryption technique using Python.

---

## 🚀 Features

* Encode (encrypt) messages
* Decode (decrypt) messages
* Custom shift values
* Handles lowercase alphabets

---

## 🛠️ Tech Used

* Python 3

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/Luck-jp/caesar-cipher-python.git

2. Navigate to the folder:
   cd caesar-cipher-python

3. Run the program:
   python main.py
